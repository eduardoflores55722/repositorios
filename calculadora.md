# Calculadora básica
Estos son los algoritmos a seguir para un programa de una calculadora básica, con operaciones de *suma*, *resta*, *multiplicación* y *división*.
## Principal
### Entrada de datos
Permitir al usuario ingresar un número **entre 1 y 5**, siguiendo la siguiente estructura:
| Entrada | Acción |
| ------------ | ------------ |
| 1 | Suma |
| 2 | Resta |
| 3 | Multiplicación |
| 4 | División |
| 5 | Salir |
### Interpretar la selección del usuario
Se sugiere utilizar lo siguiente para interpretar la selección:
``` cpp
switch (seleccion)
{
    case 1: Suma(); break;
    case 2: Resta(); break;
    case 3: Multiplicacion(); break;
    case 4: Division(); break;
    case 5: return;
    default: MenuPrincipal(); break;
}
```

**Segundo paso:** Permitir al usuario ingresar otro número. Guardarlo en una variable `n2`.
